# Go-My-Redis

Go-My-Redis 是一个使用 Golang 和 Vue.js 开发的现代化 Redis 管理工具，提供了直观的图形界面和强大的功能来管理和监控 Redis 服务器。

## 功能特点

- 🚀 现代化的用户界面
- 🔐 安全的连接管理
- 📊 实时监控 Redis 服务器状态
- 🔍 强大的键值搜索和过滤功能
- 📝 支持多种数据类型的可视化展示
- ⚡ 高性能的后端处理
- 🔄 实时数据更新

## 技术栈

### 后端
- Golang
- Gin Web 框架
- Redis 客户端库
- WebSocket 支持 [暂不支持]

### 前端
- Vue.js 3
- Naive UI 框架
- TypeScript
- Vite 构建工具
